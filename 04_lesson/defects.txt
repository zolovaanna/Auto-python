## Баг-репорт

**Название бага:** Некорректная обработка None в методе is_empty

**Описание бага:** Метод is_empty не обрабатывает None, что может привести к ошибке.

**Шаги по воспроизведению:**
1. Создать объект StringUtils.
2. Вызвать метод is_empty() с аргументом None.

**Ожидаемый результат:**
Метод должен вернуть True, так как None является пустым значением.

**Фактический результат:**
Возникает ошибка TypeError: 'NoneType' object has no attribute '__len__'

**Предлагаемое решение:**
Добавить проверку на None в методе is_empty():
Копировать
python def is_empty(self, string): """ Проверяет, является ли строка пустой.

    Args:
        string (str): Строка для проверки.

    Returns:
        bool: True, если строка пуста, False в противном случае.
    """
    return string is None or string == ""
